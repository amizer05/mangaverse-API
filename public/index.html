üìñ Manga Endpoints

GET /mangas
Lijst alle mangas met paginatie, zoeken en sorteren
Query Parameters:
limit (optioneel): aantal resultaten (default: 10, max: 100)
offset (optioneel): aantal over te slaan (default: 0)
search (optioneel): zoekterm voor title/description
sort (optioneel): sorteer veld (id, title, release_date, created_at)
order (optioneel): sorteer volgorde (asc, desc)
Voorbeeld:
GET /mangas?limit=10&offset=20&search=naruto&sort=title&order=asc
Response (200):
{
  "data": [
    {
      "id": 1,
      "title": "Naruto",
      "slug": "naruto",
      "description": "A young ninja...",
      "release_date": "2002-09-21",
      "cover_image": "https://example.com/naruto.jpg"
    }
  ],
  "pagination": {
    "total": 50,
    "limit": 10,
    "offset": 20,
    "hasMore": true
  }
}
GET /mangas/:id
Haal een specifieke manga op met gerelateerde news
Response (200):
{
  "data": {
    "id": 1,
    "title": "Naruto",
    "slug": "naruto",
    "description": "...",
    "release_date": "2002-09-21",
    "cover_image": "...",
    "news": [
      { "id": 1, "title": "...", "content": "..." }
    ]
  }
}
POST /mangas üîí AUTH REQUIRED
Maak een nieuwe manga aan
Headers:
Authorization: Bearer <token>
Request Body:
{
  "title": "One Piece", // verplicht, max 255, geen cijfers
  "description": "A pirate adventure...", // verplicht, max 255
  "release_date": "1997-07-22", // verplicht, YYYY-MM-DD, niet in toekomst
  "cover_image": "https://example.com/onepiece.jpg", // optioneel, URL
  "slug": "one-piece" // optioneel, auto-generated
}
Response (201):
{
  "message": "Manga created successfully",
  "data": { ... }
}
PUT /mangas/:id üîí AUTH REQUIRED
Update een bestaande manga
Request Body:
Alle velden optioneel (alleen te updaten velden sturen)
DELETE /mangas/:id üîí ADMIN ONLY
Verwijder een manga (alleen admin)
Response (200):
{
  "message": "Manga deleted successfully"
}

üì∞ News Endpoints

GET /news
Lijst alle news met paginatie, zoeken en filteren
Query Parameters:
limit, offset, search, sort, order
category (optioneel): filter op category (action, romance, comedy, drama)
manga_id (optioneel): filter op manga ID
Voorbeeld:
GET /news?category=action&limit=5&sort=created_at&order=desc
GET /news/:id
Haal een specifieke news op
POST /news üîí AUTH REQUIRED
Maak een nieuwe news aan
Request Body:
{
  "title": "New Chapter Release", // verplicht, max 255
  "content": "Chapter 1000 is finally here...", // verplicht, min 20 chars
  "category": "action", // verplicht: action, romance, comedy, drama
  "manga_id": 1 // optioneel, foreign key naar mangas
}
PUT /news/:id üîí AUTH REQUIRED
Update een bestaande news
DELETE /news/:id üîí ADMIN ONLY
Verwijder een news (alleen admin)

üìä Status Codes

200 OK - Succesvolle request
201 Created - Resource aangemaakt
400 Bad Request - Validatie fout
401 Unauthorized - Geen/ongeldig token
403 Forbidden - Geen admin rechten
404 Not Found - Resource niet gevonden
500 Internal Server Error - Server fout
